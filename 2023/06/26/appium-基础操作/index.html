toast操作

  1. 介绍： 操作过程中弹出的立即消失的弹出框， 本质上它也是页面的一个元素。

  2. 如何获取toast：建议使用xpath查询，因为使用工具很难定位到元素。

  3. 使用显示等待查找元素： 隐式等待的下一次找元素的频率，不适合这种一闪而过的toast

  4. 补充说明

    • 新版本的appium软件，支持Uiautomator2 因此不需要安装特殊环境

1.查看应用浏览器的版本。

  • 方式一：在应用->显示系统进程->找到Android system webview版本。

    • 不同手机模拟器和真机略有不同，推荐方式二

  • 方式二：使用命令，在dos命令行中输入命令，在手机中查看

2.下载对应的版本的浏览器内核

下载地址： http://npm.taobao.org/mirrors/chromedriver/

3.根据手机浏览器版本在上面页面中找到对应的版本

https://cdn.npmmirror.com/binaries/chromedriver/2.42/notes.txt

4.解压放到固定目录。

5.程序配置”Chromdriver Binary Path”目录
  • 打开 Appium-Server Desktop 程序配置”Chromdriver Binary Path”目录为第三部中的全路径。
  • 点击 “Start Server”开启Appium Server Desktop
